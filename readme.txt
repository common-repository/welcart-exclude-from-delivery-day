This plugin makes it possible to exclude specified dates from welcart's delivery request date.

== Description ==


* This plugin makes it possible to exclude specified dates from welcart's delivery request date
* Delivery exclusion dates can be managed using the options page.
* welcartの配送希望日から指定の日付けを除外することが可能になるプラグインです。

* Procedure for Setting Exclusion Dates
   * Click on "配送希望日設定" added to the administration screen.
   * Enter the dates you wish to exclude from your desired delivery date in the "配送除外日" field.
   * Click the "更新" button to complete the setup.

* Caution
   * Use in conjunction with the welcart e-Commerce plugin.

== Installation ==

1. From the WP admin panel, click “Plugins” -> “Add new”.
2. In the browser input box, type “Welcart exclude from delivery day”.
3. Select the “Welcart exclude from delivery day” plugin and click “Install”.
4. Activate the plugin.

OR…

1. Download the plugin from this page.
2. Save the .zip file to a location on your computer.
3. Open the WP admin panel, and click “Plugins” -> “Add new”.
4. Click “upload”.. then browse to the .zip file downloaded from this page.
5. Click “Install”.. and then “Activate plugin”.
